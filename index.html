GIT

Do konsoli logujemy się tylko raz przy użyciu komend:

  • git config --global user.name"Imię i Nazwisko"- ze względu na spację między imienim i nazwiskiem wpisujemy nasze dane w ""
  • git config --global user.email email@email.com - email nie może posiadać spacji więc nie wpisujemy go w nawiasy

Następnie upewniamy się w konsoli, że jesteśmy w folderze z naszym projektem.

Kim jestem? - ostylować później
